# lstBTC Tokenomics

- **Name**: Liquid Staked BTC
- **Symbol**: lstBTC
- **Decimals**: 18
- **Supply Model**:
  - Increases when yield is distributed.
  - Decreases when users redeem.
- **Yield Mechanism**:
  - No rebase.
  - New tokens minted via `distributeYield()`.
- **Value Peg**: 1 lstBTC = 1 BTC (via oracle pricing).
